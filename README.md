# gbot.py

## Installing to deploy
1. Installing
   ```bash
   git clone https://github.com/oss-reva/gbot.py.git
   cd gbot.py
    
   pip install .
   ```

## Installing to build
1. Installing
   ```bash
   git clone https://github.com/oss-reva/gbot.py.git
   cd gbot.py
    
   pip install -e .
   ```

## Usage
 ```bash
 gbot --token DISCORD_BOT_TOKEN
 ```
